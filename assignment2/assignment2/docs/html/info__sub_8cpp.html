Detailed Description

Node for subscribing to goal and position information and calculating average speed and remaining distance. Subscribes to the /reaching_goal/goal and /pod_vel topics and calculates average speed and remaining distance to the goal. Topics used: /reaching_goal/goal, /pod_vel.
