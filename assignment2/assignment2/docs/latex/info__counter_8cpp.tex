\doxysubsection{Detailed Description}
Node for counting the number of reached and cancelled goals and providing the count through a service. 

This node subscribes to the {\ttfamily /reaching\+\_\+goal/result} topic to receive goal status messages and updates the counters based on the received status. It also provides a service at {\ttfamily /info\+\_\+counter} to retrieve the current count of reached and cancelled goals.
